  <br>
  <br>
  This module is best used with our front-end Jekyll and Webpack-based framework <a href="https://github.com/itw-creative-works/ultimate-jekyll">Ultimate Jekyll</a>.

## Install
Install with npm:
```shell
npm install ultimate-jekyll-manager
```
